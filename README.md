# 🍫 Chocolate Sales Analysis Project

## 📌 Project Overview
This project analyzes chocolate sales data to uncover insights into:
- Country performance
- Salesperson productivity
- Product revenue contribution
- Monthly sales trends

The goal is to demonstrate data cleaning, exploratory data analysis (EDA), and visualization skills using Python.

---

## 🎯 Business Questions
1. Which country generates the highest revenue?
2. Which salesperson generates the highest sales?
3. Which product contributes the most revenue?
4. How do sales trends change over time?

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
